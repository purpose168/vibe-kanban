# Vibe Kanban

> 一款面向开发者的可视化项目管理工具，可与 git 仓库和 Claude Code、Amp 等编码代理集成。

## 快速开始

无需安装即可立即运行 Vibe Kanban：

```bash
npx vibe-kanban  # 使用 npx 运行 Vibe Kanban
```

这将在本地启动应用程序并自动在浏览器中打开。

## 什么是 Vibe Kanban？

Vibe Kanban 是一款专为开发者设计的现代项目管理工具。它通过看板式任务管理帮助您组织编码项目，同时提供与 git 仓库和 AI 编码代理的强大集成。

### ✨ 核心功能

**🗂️ 项目管理**
- 将 git 仓库添加为项目（现有或新建）
- 自动 git 集成和仓库验证
- 跨所有文件的项目搜索功能
- 每个项目的自定义设置和开发脚本

**📋 任务管理**
- 使用看板式面板创建和管理任务
- 任务状态跟踪（待办、进行中、已完成）
- 丰富的任务描述和注释
- 使用多个 AI 代理执行任务

**🤖 AI 代理集成**
- **Claude**：高级 AI 编码助手
- **Amp**：强大的开发代理
- **Echo**：简单的测试/调试代理
- 创建任务并立即启动代理执行
- 后续任务执行用于迭代开发

**⚡ 开发工作流**
- 为每个任务尝试创建隔离的 git worktree
- 查看代理所做更改的差异
- 将成功的更改合并回主分支
- 重新定位任务分支以保持最新
- 手动文件编辑和删除
- 集成开发服务器支持

**🎛️ 开发者工具**
- 从文件系统浏览和验证 git 仓库
- 在您首选的编辑器中打开任务 worktree（VS Code、Cursor、Windsurf、IntelliJ、Zed）
- 实时执行监控和进程控制
- 单独或一次性停止所有运行中的进程
- 任务完成的声音通知

## 工作原理

1. **添加项目**：导入现有 git 仓库或创建新仓库
2. **创建任务**：定义需要构建或修复的内容
3. **AI 执行**：让编码代理在隔离环境中处理您的任务
4. **查看更改**：使用 git diff 查看具体修改内容
5. **合并结果**：将成功的更改合并到您的主代码库中

## 核心功能

Vibe Kanban 提供完整的项目管理体验，具有以下关键功能：

**项目仓库管理**
- 管理编码项目的完整 CRUD 操作
- 自动 git 仓库检测和验证  
- 初始化新仓库或导入现有仓库
- 项目范围的文件搜索功能

**任务生命周期管理**
- 创建、更新和删除具有丰富描述的任务
- 通过可自定义的状态工作流跟踪任务进度
- 一键创建任务并立即执行 AI 代理
- 任务尝试跟踪，包含详细的执行历史

**AI 代理执行环境**
- 隔离的 git worktree 用于安全的代码实验
- 实时执行监控和活动日志记录
- 进程管理，可停止单个或所有进程
- 支持后续执行以迭代解决方案

**代码变更管理**
- 查看任务执行期间所有更改的详细差异
- 分支状态监控，跟踪与主分支的分歧
- 一键将成功的更改合并回主分支
- 自动重新定位以保持任务分支最新
- 手动文件删除和清理功能

**开发集成**
- 直接在您首选的代码编辑器中打开任务 worktree
- 启动和管理开发服务器以测试更改
- 浏览本地文件系统以添加新项目
- 服务可用性的健康监控

## 配置

Vibe Kanban 通过其配置系统支持自定义：

- **编辑器集成**：选择您首选的代码编辑器
- **声音通知**：自定义完成声音
- **项目默认值**：设置默认的设置和开发脚本

## 技术架构

- **后端**：使用 Axum Web 框架的 Rust
- **前端**：使用 TypeScript 的 React
- **数据库**：用于本地数据存储的 SQLite
- **Git 集成**：用于仓库管理的原生 git 操作
- **进程管理**：基于 Tokio 的异步执行监控

## 要求

- Node.js（用于 npx 执行）
- Git（用于仓库操作）
- 您首选的代码编辑器（可选，用于打开任务 worktree）

## 支持的平台

- Linux x64
- Windows x64
- macOS x64（Intel）
- macOS ARM64（Apple Silicon）
